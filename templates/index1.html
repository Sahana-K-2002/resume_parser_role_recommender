About the project

The system is built using HTML, CSS, and JavaScript for the front-end, Flask framework for the back-end, PyParser for resume parsing and Classification is used as the machine learning model.

The user interface of the system is designed using HTML and CSS while funtionality is added using JavaScript. Flask framework is used to handle user requests and responses, as well as to communicate with the classification ML model. The resume parser component of the system is built using PyParser, which extracts relevant information from resumes such as work experience, education, and skills. This information is used by the classification ML model to make job role recommendations.

It utilizes the classification model of machine learning that analyzes job descriptions and resumes to match candidates with appropriate job roles. The ML model is trained based on a dataset of job descriptions and corresponding skills, enabling it to recognize patterns and make accurate recommendations.
